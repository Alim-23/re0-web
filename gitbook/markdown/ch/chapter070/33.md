# 『向着魔都进发』

------

——使用能使用的东西，夺回了佛拉基亚皇帝的宝座。

城塞都市瓜拉尔自不必说，还包括『修德拉克之民』、兹克尔・奥斯曼『二将』以及其他因菜月・昴获得的资源。

没错，阿贝尔在确认昴的时候同意了。

「——事情好像谈妥了。」

看到会议的结论后，普莉希拉嘟囔着，用打开的扇子遮住了自己的嘴角。

正如她所说，会议显示出了一套统一的意见。共有胜利条件，决定今后的方针，决定了下一个目的地和目标。

还有就是——、

「如果少数精锐的人走向混沌世界，那之后该怎么办？现状是，我和阿贝尔，还有塔莉塔和米迪娅姆……」

「——关于那个，我可以给你一个吗？」

「阿尔？」

在决定下一个目的地——魔都混沌框架的最终成员时，为了选定人员的昴从旁边插嘴了。

他用一只手臂举手，用那只举起的手在自己脖子后面挠着

「公主，我可以稍微离开缰绳吗？我想和你的兄弟交往。」

「上！？」

「喂喂，你的声音反过来了，兄弟。有那么意外吗？」

听到了意外的提案，阿尔的喉咙被颠覆的昴震得发抖。

但是，即使是这样说，昴也不能认同“是啊”。

「为什么……」

「我不是说了什么吗？就是说要协助兄弟。像我这样落魄的大叔，我觉得也能在某个地方派上用场。」

「你那么认真吗？」

对于耸耸肩膀的阿尔的回答，昴回想起了会议前和他的对话，对于遵守在那里夹杂着的对话内容的态度感到很吃惊。

被鼓励的是事实，被鼓励的是事实，被鼓励的是事实，被鼓励的是事实。

但是，我没想到他是认真地跟我说“我会协助你的”。

「啊，凝固了怎么办？我的提议那么感动吗？」

「……不，我没听说过，如果跨越了最初的惊讶，阿贝尔会很强，我和阿贝尔的战斗力比也很差。」

「如果被期待的话就不好了，但是比那个亚马逊姐姐要弱很多！」

用手指着塔莉塔，堂堂正正地承担了阿尔可怜的事情。

实际上，塔莉塔的实力在『修德拉克之民』中也是屈指可数的，但在目睹了他与『九神将』的战斗后，增强了不可靠的战斗力。

不管怎么说——、

「但是，能这么说我本身就很开心。我会理解你的心情的。」

「嗯，别在意，兄弟……是吗？只有心情吗？难道说，这是郑重拒绝吗？这次是因为没有缘分吧？」

对昴的回答充满疑问的阿尔，肯定是接受方法。

申请本身真的很开心，但是这里是佛拉基亚帝国。说起来，阿尔的参战就交给了弗洛普和米迪娅姆以上——

「——阿尔」

「和，公主」

在做出这样结论的昴旁边，突然有一个美声叫了阿尔的名字。

声音的主人自不必说，就是堂堂正正地端坐在圆桌上的普莉希拉・跋利耶尔。她眯着红眼睛，眼睛直视着看不见感情的眼睛。

作为自己的随从者，阿尔表明了下一个行动。

「如果和妾身一起去城塞都市的话，会是你这样提出的吧。你说，把妾身放在一边和小丑伙伴出去玩？」

「虽然我并不觉得这是一次游兴的旅行，但我是这么打算的。还是说，如果我不在的话公主会很寂寞？如果她拥抱我并挽留我的话……」

「为什么？」

「是啊。」

被他那口气所遮掩，仿佛没什么期待似的垂着头。

一边盯着阿尔穿过头盔的头顶部，普莉希拉一边微微叹气

「——最多也要跳得好。」

「哦，我知道。公主才是，就算没有我和修尔特，也不能没有节制。公主的美丽，是世界的天仙。」

「不用说，我和谁都知道我的妻子。」

「当然，世界的中心就是我的公主，普莉希拉・跋利耶尔。」

用滑稽的语调说些令人讨厌的台词，阿尔在普莉希拉面前轻浮地行礼。然后回过头来的阿尔再次向昴伸出了手说“请多关照”。

「诶，刚才的对话是最终决议？我的意见是？」

「什么呀，你不是很讨厌吗？凹进去了。还是说，我只是没注意到而已，加龄臭什么的很厉害？在一起感觉很累？」

「我没这么说。不是，为什么要决定权……」

「不，当然拒绝是自由的。但是会被拒绝吗？——是公主吧？」

一口咬住下颚，用扇子扇着脸，表示百叶草。

正是这种堂堂正正的行为，才是阿尔语言压倒性的根据。把普莉希拉决定的事情推翻，这是多么可怕的事情啊。

话虽如此，如果这关系到将来的不安，也不能不拒绝。

「普莉希拉，我是……」

「——没关系。必要的话跟我来吧。」

「阿贝尔，你不是把我当军师看待吗！」

但是，正要和普莉希拉提意见的时候，他又被从背后射击了。

盯着插嘴的阿贝尔，他也堂堂正正地挽起手臂与昴相对。和普莉希拉一样，我们也打算在这里顽固地完成自己的想法。

但是，如果那样的话，雇佣条件不一样的话，昴也只能直接上诉了。

「如果不能采纳意见的话，那是为了什么而做的军师啊。」

「不要自以为是。如果是值得倾听的意见的话也会倾听。但是，掌握决定权的是我。我不会把那个交给你。」

「不知道，你是讨厌拿走工作成果的上司吗……？」

「别让我说几遍。——成果是你的。必须这样。」

「――――」

——被赶出宝座的皇帝，有着不可小觑的智者。

想要阿贝尔的是有着这样评价的昴。他大概是这样说的吧，他不嫌为了这个而让出功劳，也不嫌有点夸张。

但是，那样的话就和积累虚构没有什么区别了。

「赝品是不行的。如果不是真的，失去的东西就无法取回。」

「——那么，我和周围的人，都试着用你的手让他接受吧。在那还没实现之前，只是抱着大望的愚昧妄言。」

「可恶的家伙会帮你的。不管是什么不良债权，我都会好好使用的。」

昴盯着阿贝尔冰冷的黑瞳回答。

即使朝向同一个方向，也不是可以原谅对方的关系。这并不是王国和帝国所处的土地的差异，而是更大的差距的理由。

只要有那个，在决定性的地方和阿贝尔可能会意见不同。

然后到了那个时候，菜月・昴——

「……难道不良债权是指我吗，公主？」

「为什么？」

对于垂下肩膀这样嘟囔的阿尔，普莉希拉只是简短地回应。

——决定了远征魔都混沌框架的成员，情况发生了变化。

「在我回来之前，把城寨城市的士兵集中起来。不要太远，我会去接触的。」

「就算是代替我的生命，我也要完成。」

受到阿贝尔的命令，单膝跪拜的兹克尔恭恭敬敬地接受了命令。

没想到，站在与强大帝国为敌的立场上的兹克尔，对他的姿态毫不犹豫，但城市的官兵并非全部都是这样。

要统一那个意思，作为一个军队统一，『将』的尽力是不可缺少的。从这个意义上来说，奇迹的存在就是拾得物。

「兹克尔先生，我觉得很辛苦……」

「不，这本来是作为‘将’的我应该发挥的作用。什么，和‘无血开城’相比，是没有什么问题的。」

「――――」

昴看着挺起胸膛、如此有力地笑着的兹克尔的话。

打出“无血开城”的旗号，呼啸成功，完全失败是昴的实绩。结果，背叛了雷姆的期待，招致了失望。

但是，看到昴的样子，兹克尔眯着眼睛

「你别灰心，你为‘无血开城’献策，阁下批准了执行。事实上，我和参谋官们没有流血，而是加入了你们的军门。」

「不流血，那是……」

如果是夸张的表达，昴看着昴受伤的绷带，哽咽着说。但是，兹克尔放松了脸颊，然后强行剥下了自己的绷带。

被血弄脏的绷带被剥下，手腕上刻着的伤口暴露出来——

「这种程度，对于军人来说是擦伤。比起这个，你应该更自豪。因为你竭尽全力的结果是，为了阁下而战斗的官兵没有减少。」

「――――」

「你要完成自己的任务。——夏美小姐。」

对于这样说的兹克尔，我不知道他是否回应了合适的话。

只是，即使是不能放手称赞的成果，再次善意地考虑了能找到值得表扬的地方的兹克尔的人品，我觉得他没有死就可以了，这是成果。

然后——、

「——真的要去吗？」

「――――」

「不，我说了奇怪的话。请忘记。」

一边这样说着，一边摇头的是一双无法掩饰疲劳感的眼睛的雷姆。

雷姆拄着拐杖站在城市政府大楼入口。她的身影不是行装，而是作为治愈术师为伤员四处奔走的轻装。

雷姆留在瓜拉尔。

与前往混沌框架的昴暂时采取不同的行动。

在分别之前，对于前来送行的雷姆的一句话，昴苦笑着。即使知道这是关心我的一句话，也做不到。

「那也不全是好事，但是被我说要为你忘记什么可太累了。」

「啊，不是这个意思……」

「被勒死脖子、被折断手指、从你那里得到的一切，对我来说都是无可替代的珍贵回忆。」

「哈？」

虽然没打算揶揄，但被雷姆冷眼射穿了。

从被吹到佛拉基亚以来，回想一下冲击性大的活动，突然浮现在脑海中的记忆大多都是痛苦和痛苦的冲击。

当然，和雷姆走在一起的事，她为我着想的事，希望和喜悦的记忆，也在我的心里埋下了很多。

——最痛苦的记忆也藏在了同一个地方。

「没有紧张感……真的没问题吗？」

「一直紧张也很累吧。……被人说没关系的话，会有各种各样的想法。如果可以的话，我想一直在你身边。」

「哈」

「冷淡的回答！不，只是不被无视罢了……」

虽然被无情地伤害，但告诉雷姆的话是昴的真心话。

把雷姆留在瓜拉尔，在即将到来之前都有很大的烦恼。老实说，现在这个瞬间我也想撤回前言，抓住她的手腕带走她。

在菜月・昴的手够得到的地方，如果能保护她不受雷姆的苦难、火星等各种坏事的伤害，那该有多好呢。

「其实，我想把你和我一直用不可分割的绳子绑起来。」

「你是认真说的吗……？」

「比较」

「――――」

终于，连“哈？”也不能说了。

当然，我知道说了就会被拒绝，所以只是在废品的基础上说说而已。如果一直把自己和雷姆用绳子系在一起，随时都能知道安危的话就最好了。

根据情况的不同，也可以使用在普雷阿迪斯监视塔觉醒的力量——

「『柯尔・雷欧尼斯』也不太清楚使用方便性……」

在普雷阿迪斯监视塔中发现的昴的新的权能——“柯尔・雷欧尼斯”——那是以昴为中心，能够模糊地把握我方位置和状况的力量。另外，昴也承担了伙伴们的负担，也有能保持万无一失的一面。

或者，如果使用这种力量的话，就可以承担雷姆不自由的脚的负担，让她精神饱满地在山野中奔跑。

这样做的话，要是去了拿不到的地方就麻烦了——。

「――――」

「为什么突然露出那么失落的表情？」

「不，讨厌自己」

被雷姆指出，昴用手捂着自己萎靡的脸叹息。

我想了。因为雷姆的脚不方便，所以她来到了离不开昴的地方。无法逃脱的状态是昴和雷姆无法结束关系的原因之一。

雷姆没有万全，我觉得自己很幸运。

「就是这样，雷姆也不能相信我。」

无论什么时候，昴都只想着自己。

我想变得更温柔。我想成为温柔、聪明、坚强的存在。为了大家而努力的自己，在塔上重新审视，还不够。

——雷姆所相信的，无法取回菜月・昴。

「那个……？」

「不好意思，有点伤感。」

「老师，是什么？」

「别在意。这是和我的老家……只有阿尔才能沟通的措辞。」

抱着肩膀，雷姆微微皱着眉头回答着昴。但是，她揉了揉眼睛，消除了皱纹，好像把内心萌生的感情推后一样。

各种各样的，在雷姆中也有纠葛吧。就连这样来送我，她也应该有自己的想法。

「我不打算空出很长时间。因为没有你，我就无法忍受。」

「……又是这样。」

「嗯，我是真心的……如果心情不好的话，我会尽量控制。」

「不想放弃吧。」

被目不转睛地看着，昴放下肩膀露出反省的姿势。

当然，我不想伤害雷姆的心情，也不想让她有不愉快的回忆。但是，作为那个溢出，是对昴中的雷姆的强烈的感情。

只是，让雷姆失望的昴说什么都是。

「加油，我马上就回来。我打算带好报告回去。」

「……是的。我很期待阿贝尔、米迪娅姆、塔莉塔。」

「阿尔姑且不论，对我来说。」

「――――」

为了慎重起见问了一下，雷姆的视线温度还是没有变。

但是，对于这个回答昴却露出了非常无情的表情。雷姆沉默了一会儿，像放弃了一样微微地吐了一口气

「如果说我不相信你，那就好很多了。」

「信不信没说吧……」

「邪恶的味道依然没有减弱，但问题并不在那里。」

「问题不在那里……？」

雷姆的浅蓝色眼瞳，在其中不灭的不信任之光。

今后即使是暂时的也要分开。即使知道那是由于昴在瓜拉尔的失态造成的，也希望能去除一点。

与其说是为了昴自己，不如说是为了在城市等待雷姆的心情。

「告诉我，雷姆。如果我能做到的话，我会尽我所能努力的。怎么做才能消除你的不安呢？」

「……那样的话，为什么还穿着那个样子呢？」

「诶！？」

昴被自己眼睛里的雷姆这么说，俯视着自己的样子。

长长的黑发假发，为了隐藏伤口而涂上白粉，为了不让身体线条露出来而设计的装束和只不过是华丽的装饰品——

「哪里很奇怪吗……？」

「哪里都不奇怪是一个奇怪的地方。一点就是虽然离开了城塞城市，但是还穿着那个样子。怎么说呢？」

「不，我不是说过这是必要的吗！」

看着雷姆眼睛的温度下降，昴继续着夏美・施瓦茨的状态，发出了悲鸣般的声音。

没想到这身打扮对她的不信任起到了一定的作用，但正如雷姆所说的那样，这也是有确切的理由的。

「雷姆，我昨天也说过了，我们是这个国家邻国的人。但是，如果我的真名被大家知道的话，就会跨国给你们添麻烦。所以，有必要。不是真正的我，而是夏美・施瓦茨的存在……！」

「啊，是吗？」

「完全没有消除不信任感的回答！」

雷姆的眼角和眼睛的温度没有变化，反而更增加了不信任感。

但是，这是把昴所处的状况和所施加的作用放在天平上，认真设计的对策。

——在这个佛拉基亚帝国，不能卖“菜月・昴”的名字。

菜月・昴已经是露格尼卡王国重要人物爱蜜莉娅的骑士了。也就是说，现在所做的事情是对邻国的良好内政干涉。

「不，干涉内政也许没什么了不起的……」

不管怎样，最重要的是昴的行动结果，其责任不只是昴坦率地说，可能会给爱蜜莉娅带来麻烦。

作为她的骑士，昴发誓要帮助她成为国王，绝对不允许她做出拖后腿的举动。

「在那里，是夏美・施瓦茨。如果是夏美的名字，卖多少都没问题。夏美的话，只不过是在佛拉基亚帝国突然出现的黑发美少女罢了。」

「结束了吗？」

「还没呢！还有，作为目算……如果是夏美・施瓦茨的名字的话，我在露格尼卡的亲人可能会注意到。」

不如说，将假名限定为“夏美・施瓦茨”的最大理由就在于此。

昴就这样穿女装，自称夏美・施瓦茨，这不是佛拉基亚帝国第一次发生的事情。虽然不是在罗兹瓦尔宅邸的余兴，但是有必要的时候会有女装的机会，除了爱蜜莉娅以外的人都知道她的真实身份。

那时，自报姓名的假名也是一样的。也就是说今后，如果按照阿贝尔的想法昴的存在在帝国内获得了知名度，而扩展开来的名字是“夏美・施瓦茨”的话，爱蜜莉娅们可能会有机会知道昴他们被飞到帝国。

故——、

「我迫于需要，才穿成这个样子。」

「——。——————————我明白了。」

虽然花了相当长的时间，但得到了雷姆的理解，松了一口气。

总之，昴的女装状态现在暂时维持。我不想因为拖得太久而损害雷姆的信赖，但这也要看情况而定。

「雷姆，如果有什么困难的话，请和弗洛普先生或兹克尔先生商量。如果是男人的话，有困难的事情的话，也有米泽尔妲小姐。不要一个人承担。」

「我不理解你说的话，但我会听的。……那才是，请不要给阿贝尔先生和米迪娅姆小姐添麻烦。」

「只注意后面。」

先不说米迪娅姆和塔莉塔，也不打算对阿贝尔造成负担。

偶尔也应该打破一张凉爽的脸，满头大汗的阿贝尔也应该苦战恶斗。

而且，如果和一种注意事项惜别的话，出发的时候也会临近。

在出发前——

「雷姆，那家伙怎么了？」
